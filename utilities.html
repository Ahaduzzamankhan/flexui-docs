Utility Functions

FlexUI ships a set of small, tree-shakeable utility functions for class names, objects, strings, async, and URLs. Import only what you use.

import { cx, debounce, slugify } from 'flexui/utils';

cx — Class Names

Conditionally join CSS class names. Accepts strings, objects, arrays, and falsy values.

import { cx } from 'flexui/utils';

cx('btn', isActive && 'btn--active', { 'btn--lg': large })
// → 'btn btn--active btn--lg'

cx('card', null, undefined, 'card--dark')
// → 'card card--dark'

Object Utilities

deepClone(obj)

Returns a deep copy of an object, array, or Date. Does not clone class instances.

const copy = deepClone({ a: { b: [1, 2] } });
// copy.a !== original.a  →  true

pick(obj, keys)

Returns a new object with only the specified keys.

pick({ a: 1, b: 2, c: 3 }, ['a', 'c'])  // → { a: 1, c: 3 }

omit(obj, keys)

Returns a new object with the specified keys removed.

omit({ a: 1, b: 2, c: 3 }, ['b'])  // → { a: 1, c: 3 }

shallowEqual(a, b)

Returns true if two objects have the same top-level keys and values.

shallowEqual({ x: 1 }, { x: 1 })  // → true

Function Utilities

debounce(fn, delay)

Returns a debounced version of fn. Call .cancel() to clear the pending timer.

const debouncedSearch = debounce(fetchResults, 300);
debouncedSearch.cancel();

throttle(fn, interval)

Returns a throttled version of fn — calls at most once per interval milliseconds.

const throttledScroll = throttle(handleScroll, 100);

once(fn)

Returns a function that invokes fn exactly once and returns the same result on subsequent calls.

const init = once(heavySetup);

memoize(fn)

Caches results by serialized arguments. Best for pure functions with serializable inputs.

const memoFib = memoize((n) => n <= 1 ? n : memoFib(n-1) + memoFib(n-2));

URL Utilities

joinPaths(...parts)

joinPaths('/blog', '2025', 'my-post')  // → '/blog/2025/my-post'

parseQuery(search)

parseQuery('?q=flex&page=2')  // → { q: 'flex', page: '2' }

stringifyQuery(params)

stringifyQuery({ q: 'flex', page: 2 })  // → 'q=flex&page=2'

Async Utilities

sleep(ms)

Returns a promise that resolves after ms milliseconds.

await sleep(500); // pause 500ms

retry(fn, options)

Retries an async function with exponential backoff until it resolves or exhausts attempts.

const data = await retry(
  () => fetch('/api/data').then(r => r.json()),
  { attempts: 3, delay: 500 }
);
OptionDefaultDescription
attempts3Maximum number of tries
delay500Base delay in ms (doubles each retry)
