Hooks

Hooks let functional components use state, side effects, and other framework features without writing a class. All hooks follow the same rules as React hooks.

Rules of Hooks
Only call hooks at the top level of a function component or custom hook. Never inside conditions, loops, or nested functions.

useState

Adds local state to a functional component. Returns the current value and a setter function.

const [state, setState] = useState(initialState)
import { useState } from 'flexui';

function Toggle() {
  const [on, setOn] = useState(false);
  return createElement('button', { onClick: () => setOn(v => !v) },
    on ? 'ON' : 'OFF'
  );
}

// Functional initializer (runs only once)
const [items, setItems] = useState(() => loadFromStorage());

useReducer

An alternative to useState for complex state logic. Accepts a reducer and an initial state.

const [state, dispatch] = useReducer(reducer, initialArg, init?)
import { useReducer } from 'flexui';

function reducer(state, action) {
  switch (action.type) {
    case 'increment': return { count: state.count + 1 };
    case 'reset':     return { count: 0 };
    default:          return state;
  }
}

function Counter() {
  const [state, dispatch] = useReducer(reducer, { count: 0 });
  return createElement('div', null,
    createElement('p', null, state.count),
    createElement('button', { onClick: () => dispatch({ type: 'increment' }) }, '+'),
    createElement('button', { onClick: () => dispatch({ type: 'reset' }) }, 'Reset'),
  );
}

useEffect

Runs a side effect after the browser has painted. Return a cleanup function to run on unmount or before the next effect.

useEffect(effect, deps?)
import { useEffect } from 'flexui';

// Runs after every render
useEffect(() => { document.title = title; });

// Runs once on mount
useEffect(() => {
  const id = setInterval(tick, 1000);
  return () => clearInterval(id); // cleanup
}, []);

// Runs when `userId` changes
useEffect(() => {
  fetchUser(userId).then(setUser);
}, [userId]);

useLayoutEffect

Identical to useEffect but fires synchronously after DOM mutations and before the browser paints. Use sparingly — prefer useEffect.

useLayoutEffect(() => {
  // Measure DOM before paint
  const { height } = ref.current.getBoundingClientRect();
  setHeight(height);
}, []);

useRef

Returns a mutable ref object whose .current property persists across renders without triggering a re-render. Commonly used to hold a DOM node reference.

const ref = useRef(initialValue)
import { useRef, useEffect } from 'flexui';

function FocusInput() {
  const inputRef = useRef(null);

  useEffect(() => { inputRef.current.focus(); }, []);

  return createElement('input', { ref: inputRef });
}

useMemo

Memoizes a computed value. Re-computes only when dependencies change.

const value = useMemo(() => expensiveCalc(a, b), [a, b]);
Only memoize expensive work
Don't wrap every computation in useMemo. The overhead of tracking dependencies adds up — only use it for genuinely expensive operations or to preserve reference identity.

useCallback

Returns a memoized callback function. Equivalent to useMemo(() => fn, deps).

const handleClick = useCallback(() => {
  doSomething(id);
}, [id]);

useContext

Reads and subscribes to a context value. The component re-renders when the context value changes.

const value = useContext(MyContext);
import { useContext } from 'flexui';
import { ThemeCtx } from '../context/theme.js';

function ThemeLabel() {
  const theme = useContext(ThemeCtx);
  return createElement('span', null, `Theme: ${theme}`);
}

useId

Generates a stable, unique ID that is consistent between server and client renders. Useful for accessible form labels.

import { useId } from 'flexui';

function FormField({ label }) {
  const id = useId();
  return createElement('div', null,
    createElement('label', { htmlFor: id }, label),
    createElement('input', { id })
  );
}

useImperativeHandle

Customizes the instance value exposed to parent components via ref. Use with forwardRef.

useImperativeHandle(ref, createHandle, deps?)
const Dialog = forwardRef(function(props, ref) {
  const [open, setOpen] = useState(false);

  useImperativeHandle(ref, () => ({
    show: () => setOpen(true),
    hide: () => setOpen(false),
  }), []);

  return open ? createElement('dialog', { open }, props.children) : null;
});

// Parent calls dialogRef.current.show()
