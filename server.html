Dev & Production Server

FlexUI provides a fully-featured HMR development server and a production SSR HTTP server, both accessible via the CLI or the programmatic API.

On this page

Dev Server

The dev server handles esbuild rebuilds, WebSocket HMR, and static asset serving. Start it via the CLI:

flexui dev --port 3000 --open

Or programmatically:

import { startDevServer } from 'flexui/server';
import { loadConfig } from 'flexui/build';

const config = await loadConfig(process.cwd());
const { server, close } = await startDevServer({ ...config, port: 3000 });

console.log('Dev server running on http://localhost:3000');

// Graceful shutdown
process.on('SIGINT', () => close());

startDevServer API

OptionTypeDescription
portnumberHTTP port (default: 3000)
hoststringInterface to bind (default: 'localhost')
entrystringClient entry point
publicDirstringStatic assets directory

Returns: { server: http.Server, wss: WebSocketServer, close: () => void }

Production Server

The production SSR server handles incoming HTTP requests, calls getServerSideProps, renders to HTML, and streams the response. Start via CLI after a build:

flexui build --ssr
flexui start --port 8080

startProdServer API

import { startProdServer } from 'flexui/server';

const { server } = await startProdServer({
  distDir:    'dist',
  port:       8080,
  host:       '0.0.0.0',
});
OptionTypeDescription
distDirstringPath to the build output directory
portnumberHTTP port
hoststringBind address. Use '0.0.0.0' to accept external connections.

Static Files

Both servers serve files from publicDir with correct MIME types and cache headers:

File typeCache-Control header

Custom Middleware

If you need to add custom request handling (authentication, rate-limiting, logging), integrate FlexUI's SSR renderer into your own HTTP server:

import http from 'node:http';
import { renderToString } from 'flexui/server';
import App from './src/App.js';

http.createServer(async (req, res) => {
  // Your middleware logic here
  if (req.url === '/healthz') {
    res.end('ok'); return;
  }

  const html = await renderToString(createElement(App, { url: req.url }));
  res.setHeader('Content-Type', 'text/html');
  res.end(`<!DOCTYPE html><html><body>${html}</body></html>`);
}).listen(3000);
