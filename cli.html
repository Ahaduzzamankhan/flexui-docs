FlexUI v1.1.3

CLI Reference

The flexui CLI manages your entire project lifecycle — scaffolding, development, building, and deployment.

On this page
flexui create flexui dev flexui build flexui start flexui generate

flexui create

Scaffold a new FlexUI project from a template.

flexui create <name> [options]
OptionDefaultDescription
-t, --template <name>defaultProject template: default, ssr, ssg
--tsfalseInclude TypeScript configuration
 
# Default SPA
flexui create my-app

# SSR project
flexui create my-app -t ssr

# SSG with TypeScript
flexui create my-site -t ssg --ts

flexui dev

Start the HMR development server. Changes to source files are reflected in the browser instantly.

flexui dev [options]
OptionDefaultDescription
-p, --port <n>3000Port to listen on
-H, --host <host>localhostNetwork interface to bind
--openfalseOpen browser automatically on start
 
flexui dev --port 4000 --open
HMR via WebSocket
The dev server communicates with the browser over WebSocket. If you run behind a proxy, make sure it forwards the Upgrade: websocket header.

flexui build

Bundle the project for production using esbuild. Outputs minified, tree-shaken files to outDir.

flexui build [options]
OptionDescription
--ssgPre-render all pages at build time (Static Site Generation)
--ssrEmit a server bundle for runtime SSR
 
# Standard CSR build
flexui build

# SSG (generates static HTML for each page)
flexui build --ssg

# SSR (builds both client + server bundles)
flexui build --ssr

Build Output

dist/
├── index.html
├── asset-manifest.json
├── _chunks/
└── static/
    └── js/
        ├── main.[hash].js
        └── vendor.[hash].js

flexui start

Start the production SSR HTTP server. Requires a prior flexui build --ssr.

flexui start [options]
OptionDefaultDescription
-p, --port <n>3000Port to serve on
 
flexui build --ssr && flexui start --port 8080
Static hosting for SSG
SSG output does not need flexui start. The dist/ folder can be deployed directly to any static host (Netlify, Vercel, GitHub Pages, S3).

flexui generate

Generate SEO and infrastructure files from your route manifest and config.

flexui generate <type>
TypeOutputDescription
sitemapdist/sitemap.xmlXML sitemap from all routes in pagesDir
robotsdist/robots.txtrobots.txt from config
 
flexui generate sitemap
flexui generate robots
