Router

FlexUI ships with a file-based router and a client-side history router. Drop files into /pages and routes are generated automatically.

File-Based Routing

The FlexUI file router scans the pagesDir (default: pages/) and builds a route manifest automatically.

FileRoute
pages/index.js/
pages/about.js/about
pages/blog/index.js/blog
pages/blog/[slug].js/blog/:slug
pages/docs/[...path].js/docs/* (catch-all)
pages/404.jsNot-found fallback

Dynamic Segments

Wrap a segment in square brackets to make it dynamic. Access the value via useParams.

pages/blog/[slug].js 
import { createElement } from 'flexui';
import { useParams } from 'flexui/router';

export default function BlogPost() {
  const { slug } = useParams();
  return createElement('article', null,
    createElement('h1', null, `Post: ${slug}`)
  );
}

// SSG — tell the router which slugs exist
export async function getStaticPaths() {
  const posts = await fetchAllPosts();
  return { paths: posts.map(p => ({ params: { slug: p.slug } })) };
}

Link & NavLink

Link renders an accessible anchor that uses client-side navigation. NavLink adds an active class automatically when the href matches the current path.

import { Link, NavLink } from 'flexui/router';

// Basic link
createElement(Link, { href: '/about' }, 'About');

// Prefetch on hover (default: true)
createElement(Link, { href: '/blog', prefetch: true }, 'Blog');

// NavLink — adds class="active" when path matches
createElement(NavLink, { href: '/docs', activeClass: 'nav--active' }, 'Docs');

useRouter

Returns the router instance with the current pathname, query, and navigation methods.

import { useRouter } from 'flexui/router';

function SearchBar() {
  const router = useRouter();

  return createElement('input', {
    onKeyDown: (e) => {
      if (e.key === 'Enter')
        router.push(`/search?q=${e.target.value}`);
    }
  });
}

Router API

Property / MethodDescription
router.pathnameCurrent URL pathname (e.g. /blog/hello)
router.queryParsed query string as a plain object
router.push(url)Navigate to a new URL, adds to history
router.replace(url)Navigate without adding a history entry
router.back()Go back one step in history
router.forward()Go forward one step in history
router.prefetch(path)Pre-load a route's component bundle

useParams

Returns an object of dynamic route segments for the current page.

// URL: /blog/hello-world
const { slug } = useParams(); // { slug: 'hello-world' }

useSearchParams

Returns a parsed object of the current URL query string.

// URL: /search?q=flexui&page=2
const { q, page } = useSearchParams(); // { q: 'flexui', page: '2' }

Programmatic Navigation

const router = useRouter();

// Navigate after form submit
async function handleSubmit(e) {
  await saveData(formData);
  router.push('/dashboard');
}

// Replace current entry (login redirect)
if (!user) router.replace('/login');

Prefetching

Call router.prefetch() or set prefetch on <Link> to preload a route's JavaScript bundle before the user navigates. This makes navigation feel instant.

// Prefetch on button hover
createElement('button', {
  onMouseEnter: () => router.prefetch('/dashboard'),
  onClick: () => router.push('/dashboard'),
}, 'Go to Dashboard');
