Built-in Components

FlexUI ships a small set of utility components in flexui/components for common UI patterns: error boundaries, suspense, portals, modals, conditional rendering, and list rendering.

ErrorBoundary

Catches JavaScript errors thrown anywhere in the child component tree and renders a fallback UI instead of crashing the page.

import { ErrorBoundary } from 'flexui/components';

createElement(ErrorBoundary, {
  fallback: (error) => createElement('p', { style: { color: 'red' } }, error.message),
  onError:  (err, info) => logToSentry(err, info),
},
  createElement(RiskyComponent, null)
);

ErrorBoundary Props

PropTypeDescription
fallbackElement | (error) => ElementUI to render on error. Receives the error object if a function.
onError(error, info) => voidCalled when a child throws. Use for logging.

Suspense

Shows a fallback while a lazily-loaded component is being fetched. Works with lazy().

import { lazy } from 'flexui';
import { Suspense } from 'flexui/components';

const BigMap = lazy(() => import('./BigMap.js'));

createElement(Suspense, {
  fallback: createElement('div', { className: 'skeleton' }, 'Loading map…')
},
  createElement(BigMap, { center: [0, 0] })
);

Portal

Renders children into a different DOM node, outside the current component tree. Useful for modals and tooltips.

import { Portal } from 'flexui/components';

// Renders into document.body by default
createElement(Portal, null,
  createElement('div', { className: 'tooltip' }, 'I am in <body>!')
);

// Custom target (CSS selector or DOM element)
createElement(Portal, { target: '#modal-root' },
  createElement(MyModal, null)
);

Modal

A fully accessible dialog with keyboard handling (Escape to close) and portal rendering.

import { Modal } from 'flexui/components';

const [open, setOpen] = useState(false);

createElement(Modal, {
  isOpen:  open,
  onClose: () => setOpen(false),
  title:   'Confirm Delete',
},
  createElement('p', null, 'Are you sure? This cannot be undone.'),
  createElement('button', { onClick: handleDelete }, 'Delete'),
);

Modal Props

PropTypeDescription
isOpenbooleanControls visibility
onClose() => voidCalled on backdrop click or Escape key
titlestringUsed for aria-label on the dialog
classNamestringAdditional CSS class on the dialog element

Show

Conditionally renders children based on a when prop. Cleaner than inline ternaries for complex conditional trees.

import { Show } from 'flexui/components';

createElement(Show, {
  when:     isLoggedIn,
  fallback: createElement(LoginButton, null),
},
  createElement(UserAvatar, { user })
);

For

Renders a keyed list from an array. Handles the key prop automatically using the item index or a custom key function.

import { For } from 'flexui/components';

createElement(For, {
  each:   products,
  key:    (item) => item.id,
  render: (item) => createElement(ProductCard, { product: item }),
  empty:  createElement('p', null, 'No products found.'),
});

Transition

Applies CSS class transitions when a component enters or leaves the DOM.

import { Transition } from 'flexui/components';

createElement(Transition, {
  show:       visible,
  enterClass: 'fade-enter',
  leaveClass: 'fade-leave',
  duration:   300,
},
  createElement('div', { className: 'panel' }, 'Content')
);
