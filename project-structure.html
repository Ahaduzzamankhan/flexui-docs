Project Structure

Overview of the files and directories in a FlexUI project, and what each one does.

Directory Layout

my-app/
├── flexui.config.js        # Framework configuration
├── package.json
├── pages/                  # File-based routes
│   ├── _app.js             # Global layout / app shell
│   ├── _document.js        # HTML document shell (SSR/SSG)
│   ├── index.js            # Route: /
│   ├── about.js            # Route: /about
│   └── blog/
│       ├── index.js        # Route: /blog
│       └── [slug].js       # Route: /blog/:slug  (dynamic)
├── components/             # Reusable UI components
│   └── Button.js
├── public/                 # Static files (served as-is)
│   ├── index.html          # HTML entry (CSR)
│   ├── styles.css
│   └── favicon.ico
└── src/
    └── main.js             # Client-side entry point

Pages Directory

Every file inside pages/ automatically becomes a route. FlexUI's file router translates file paths to URL paths:

FileURLNotes
pages/index.js/Home route
pages/about.js/about
pages/blog/index.js/blog
pages/blog/[slug].js/blog/:slugDynamic segment
pages/docs/[...rest].js/docs/*Catch-all segment
pages/_app.js—Global layout shell

_app.js — Global Layout

Wrap every page in a shared layout by exporting a component from pages/_app.js:

pages/_app.js 
import { createElement } from 'flexui';

export default function App({ Component, pageProps }) {
  return createElement('div', { className: 'app' },
    createElement('nav', null, /* global nav */),
    createElement(Component, pageProps),
    createElement('footer', null, /* global footer */),
  );
}

Client Entry Point

src/main.js 
import { createRoot } from 'flexui';
import { FlexRouter } from 'flexui/router';
import App from '../pages/_app.js';

const root = createRoot(document.getElementById('root'));
root.render(createElement(FlexRouter, null, createElement(App, null)));

Public Directory

Files in public/ are served as static assets. Reference them from your HTML or components using root-relative paths:

<!-- public/index.html -->
<div id="root"></div>
<script type="module" src="/main.js"></script>

Build Output

dist/
├── index.html
├── asset-manifest.json     # Maps source files to hashed bundles
├── _chunks/                # Code-split chunks
├── static/
│   └── js/
│       ├── main.[hash].js
│       └── vendor.[hash].js
└── sitemap.xml             # (if generated)
